今回の修正点
・コンプレッサでレベルを持ち上げたい場合は、Makeup Gain を明示的に操作。初期値は 0 dB。
・リミッタは |x| <= ceiling の範囲で 完全に等倍、超えた分だけ Ceiling に制限。
・つまり、レベルが上がるならそれは Makeup Gain または Output Gain の結果 です。
・クレストファクター教材として、PRE / POST差の理由が追いやすくなっています。
補足
・file:// 直開き対応（外部ライブラリなし）
・CF = 20 log10(Peak / RMS)
・サイン波はおおむね 3 dB、方形波はおおむね 0 dB
・リミッタは教育用に「Hard Ceiling」寄りの実装です（Ceiling以下ユニティ保証）
・見やすさのため、スペクトラムには時間平均・周波数平滑を入れています
