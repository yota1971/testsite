使い方（ポイント）

  • Noise（White/Pink） → Start が見やすいです。
  • EQカーブ：ドラッグ（横=周波数、縦=ゲイン、Shift+縦=Q）
  • Spec 前 = source 直後（信号レベル前）
  • Spec 後 = Spec 前に EQ 理論応答を加えた表示
  • 時間平準化 は表示の時間方向スムージングです。
