指向性マイクの原理

2つの受音信号 A, B から、無指向（Omni） = (A+B)/2、双指向（Figure-8） = (A−B)/2 を作り、 出力 = a·Omni + b·Figure-8 で カーディオイド / スーパーカーディオイド / ハイパーカーディオイド を生成します。

指向性合成

カーディオイドは a=0.50, b=0.50。前方を保ち、後方を抑える基本形です。
典型例: 無指向=(1,0)、双指向=(0,1)、カーディオイド=(0.50,0.50)、スーパー≈(0.37,0.63)、ハイパー≈(0.25,0.75)

無指向は圧力そのものを見るので方向差が出にくい。
双指向は前後差（圧力勾配）を見るので、前後で極性が反転し、横で小さくなりやすい。
カーディオイドは 無指向 + 双指向 の和として説明でき、前方を保ちながら後方を抑える。
ただし実際の物理構造は 周波数依存 を持つため、理想形と完全一致はしません。

要するに、指向性の本質は「向き」ではなく、時間差 → 位相差 → 干渉 → 線形結合です。
