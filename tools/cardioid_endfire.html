サブウーファー Cardioid / End-fire 比較ツール（カーソル読取り付き・検算しやすい版）

2台配列の Cardioid（gradient） と End-fire を比較します。
画面上でパラメーターを動かしながら、配列模式図・ポーラー・後方減衰・前後レベル・ベクトル合成 を同時に見られるダッシュボードです。
カーソル読み取り と 数値検算パネル を追加しています。外部ライブラリ不使用、単体HTML、file:// 対応です。

設定 / プリセット

モデルの約束： 前方 = 0°（右）、後方 = 180°（左）、前ユニットは前側 x=0、後ユニットは後側 x=-d。
Cardioid は 後ユニットを逆相 + d/c 遅延、End-fire は 前ユニットを d/c 遅延 としています。
表示用 dB は教材向けに -60 dB を下限、後方減衰表示は 0〜40 dB に制限しています。
