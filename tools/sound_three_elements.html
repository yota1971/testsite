音の三要素

周波数・音量・音色のメカニズムを学ぶ

1. 周波数（Frequency）

周波数は1秒間に波が振動する回数を示します。単位はHz（ヘルツ）です。 周波数が高いほど高い音になり、低いほど低い音になります。

周波数を調整してください 440 Hz
出力音量 80%
スタンバイ
基準周波数（A4）: 440 Hz は国際標準の基準音です。 ピアノの中央のラ（A）の音がこの周波数です。
音域の参考値:
• 低音域: 20 Hz ～ 250 Hz
• 中音域: 250 Hz ～ 4 kHz
• 高音域: 4 kHz ～ 20 kHz

2. 音量（Amplitude）

音量は振幅（波の上下の大きさ）で決まります。単位はdB（デシベル）です。 0 dB を基準として、-∞ dB （無音）から計測されます。

振幅（リニア）を調整 0.7
音圧レベル（dB） -3.1 dB
スピーカー出力 80%
スタンバイ
デシベルの計算式:
dB = 20 × log₁₀(Amplitude / 1.0)

例: Amplitude = 0.7 の場合
dB = 20 × log₁₀(0.7) ≈ -3.1 dB
実世界の音圧レベル参考値:
• 無音: -∞ dB
• 囁き声: 30 dB
• 通常会話: 60 dB
• 交通騒音: 80 dB
• ロックコンサート: 120 dB
• 痛みを感じる閾値: 130 dB

3. 音色（Timbre）

音色は同じ周波数の音でも、楽器によって異なる聴こえ方です。 これは基本周波数に加えて、倍音（ハーモニクス）がどのように含まれているかで決まります。

倍音成分
各波形の倍音構成:

正弦波 (Sine):
基本周波数のみ（倍音なし）→ 最も単純な音

矩形波 (Square):
基本 + 3倍 + 5倍 + 7倍... (奇数倍音のみ)
強い高音成分 → 明るく硬い音

のこぎり波 (Sawtooth):
基本 + 2倍 + 3倍 + 4倍... (すべての倍音)
最も多くの倍音 → 豊かで複雑な音

三角波 (Triangle):
基本 + 3倍 + 5倍... (奇数倍音のみ、矩形波より弱い)
実楽器との関係:
実際の楽器の音は複数の周波数成分を持っています。 ピアノ、ギター、トランペットなど、それぞれの倍音構成が その楽器固有の「音色」を作り出しています。
